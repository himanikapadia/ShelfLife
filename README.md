# 📦 ShelfLife

**ShelfLife** is an interactive, terminal-based inventory management application built in Python. The project is designed to help users track product lifecycles, categorize inventory items, monitor low stock counts, and receive smart alert notices before items expire.

Instead of writing a single, sprawling script, **ShelfLife** transitions theoretical Object-Oriented Programming (OOP) concepts into a production-ready, modular system architecture.
